# BilibiliMovieControl

一款针对 Bilibili（以下称 B 站）视频选集连播时自动切换剧集的浏览器插件。通过智能像素级分析与灵活的存档机制，让你的观看体验真正实现“无人值守”。

---

## 🚀 核心功能

### 1. 模式选择
本插件提供两种切集逻辑，满足不同场景需求：

* **智能帧分析：** 采用像素级帧分析技术，实时检测视频末尾是否出现黑屏或静止画面。符合条件即自动跳转到下一 P。
    * **便捷性：** 用户设置起始点即可。
* **手动模式：** 切换后将显示时间输入框，允许用户自定义精确的切集时间点（需点击保存）。

### 2. OP / 先导跳过
针对片头较长且内容固定的合集，可手动设置“跳过区间”。当播放进度进入该区间时，插件将自动跨越到设定的正片起始时间点。

### 3. 多重存档机制
* **最近播放（自动存档）：** 后台自动记录最近播放的 **20 条** 视频配置。当你再次打开相同合集时，插件会自动恢复之前的 OP 跳过和切集设置。
* **手动存档：** 支持最多 **20 条** 长期手动配置。点击“存档”会同步保存当前的标题、URL 以及所有时间点参数。再次点击列表记录即可一键还原。
* **快速访问：** 插件 Popup 弹窗界面会优先显示最近的 **2 条** 记录，方便快速切换。


---

## ⚠️ 适用范围说明

> [!IMPORTANT]
> **本插件仅针对 B 站用户上传的“合集”生效。**
> 即播放器右侧显示有选集列表的页面。对电影、番剧正片、纪录片或普通单个视频页面无效。
