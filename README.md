## BilibiliMovieControl

一款针对Bilibili（以下称B站）视频选集连播时自动切换剧集的浏览器插件。

### 条件

适用于B站用户上传的合集。

**只针对用户上传合集，对其他无效。**


## 功能

 **1. 模式选择**

本插件默认自动末尾切集，可手动切换。

自动切换采用帧分析，是否为剧集末尾黑屏，符合条件开始跳转到下一P。

* 为了防止过渡使用cpu，一般在视频的最后 15% 阶段才会开始分析。

* 自动模式末尾切集无需保存。

* 切换手动模式后，设置切集的时间框会显示，需要保存。

**2. op，先导跳过**

手动设置，在片头，一般比较固定，看情况设置。

**3. 最近播放**

自动保存最新播放的两条视频记录。

**4. 手动存档**

* 存档最多三条记录，
* 保存时会自动保存，设置的op跳过和切集的时间点。
* 点击存档 会自动设置之前保存的op跳过和切集的时间点数据。
